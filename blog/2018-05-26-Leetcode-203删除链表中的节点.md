删除链表中等于给定值 **val** 的所有节点。

**示例:**

```
输入: 1->2->6->3->4->5->6, val = 6
输出: 1->2->3->4->5
```

**解题思路**

```python
class Solution:
    def removeElements(self, head, val):
        """
        :type head: ListNode
        :type val: int
        :rtype: ListNode
        """
        while head != None and head.val == val:
            delNode = head
            head = delNode.next
            delNode = None

        if head == None:
            return head

        cur = head
        while cur.next != None:
            delNode = cur.next
            if delNode.val == val:
                cur.next = delNode.next
                delNode = None
            else:
                cur = cur.next

        return head
```
